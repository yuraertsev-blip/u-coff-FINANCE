# U-Coffee (Ю кофе) - SPA для учета финансов

Mobile-first Single Page Application (SPA) для учета финансов кофейни "Ю кофе". 

Приложение не требует установки серверной части и работает полностью локально, используя `LocalStorage` для хранения данных. Разработано с фокусом на минимализм, премиальный дизайн (Glassmorphism, светлая тема) и высокую скорость работы.

## Особенности
- **Ввод данных:** Интерактивный календарь, блок доходов (выручка) и удобная таблица для расходов (до 15 строк в день).
- **Автозаполнение:** При вводе названия расхода система предлагает варианты из истории.
- **Аналитика:** Анализ доходов и расходов за выбранный период (по умолчанию - текущий месяц), визуальное отображение на столбчатой диаграмме и подробный отчет по категориям расходов.
- **Настройки:** Управление справочником категорий. Изменения тут же отражаются в списках выбора в разделе "Ввод данных".
- **Интуитивный UI/UX:** Автоматическое форматирование денежных сумм (например: `10 000`) и блокировка ввода текста в цифровые поля.

## Технологии
- Ванильный JavaScript (ESM)
- HTML5 / CSS3 (кастомный дизайн без тяжеловесных фреймворков)
- `LocalStorage` (хранение базы данных прототипа)
- Иконки: Lucide SVG (встроены)
- Шрифты: Google Fonts (Outfit)

## Установка и запуск

Поскольку проект не требует сборки через Node.js (Vite/Webpack), его можно запустить любым простым веб-сервером.

### Быстрый старт (Python)
Если у вас установлен Python:
```bash
cd u_coffee_app
python3 -m http.server 8000
```
Затем откройте в браузере `http://localhost:8000`

### Быстрый старт (Live Server)
Если вы используете VS Code, просто откройте папку `u_coffee_app` и нажмите **"Go Live"** (при наличии расширения Live Server).

## Безопасность и архитектура
- Исходный код написан согласно концепции "одна ответственность" (разделены функции форматирования, состояния, и рендеринга видов).
- Безопасная обработка вводимых данных: экранируются ненужные символы, все числовые данные приводятся к валидным целым числам перед сохранением.
- Все данные лежат локально в браузере пользователя, что гарантирует полную конфиденциальность учета.
